## installation

official QT releases are available [here](https://download.qt.io/official_releases/)

download official QT creator [here](https://download.qt.io/official_releases/qtcreator/)

pyside6 pip installation command

```
pip install pyside6
```

after installation check by importing pyside6

```python
import pyside6
```
